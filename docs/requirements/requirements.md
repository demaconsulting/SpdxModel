# SpdxModel Library Requirements

## Functional Requirements

| ID | Title |
| :- | :---- |
| REQ-FUNC-001 | The library shall support deserializing SPDX 2.2 JSON documents. |
| REQ-FUNC-002 | The library shall support deserializing SPDX 2.3 JSON documents. |
| REQ-FUNC-003 | The library shall support serializing SPDX documents to JSON format. |
| REQ-FUNC-004 | The library shall support SPDX document creation information. |
| REQ-FUNC-005 | The library shall support SPDX packages. |
| REQ-FUNC-006 | The library shall support SPDX files. |
| REQ-FUNC-007 | The library shall support SPDX snippets. |
| REQ-FUNC-008 | The library shall support SPDX relationships. |
| REQ-FUNC-009 | The library shall support SPDX annotations. |
| REQ-FUNC-010 | The library shall support SPDX checksums with multiple algorithms. |
| REQ-FUNC-011 | The library shall support SPDX external references. |
| REQ-FUNC-012 | The library shall support SPDX external document references. |
| REQ-FUNC-013 | The library shall support SPDX extracted licensing information. |
| REQ-FUNC-014 | The library shall support SPDX package verification codes. |
| REQ-FUNC-015 | The library shall provide utilities for manipulating SPDX relationships. |
| REQ-FUNC-016 | The library shall support retrieving root packages from an SPDX document. |
| REQ-FUNC-017 | The library shall support deep copying of SPDX elements. |
| REQ-FUNC-018 | The library shall support validation of SPDX elements. |

## Quality Requirements

| ID | Title |
| :- | :---- |
| REQ-QUAL-001 | The library shall support .NET 8 runtime. |
| REQ-QUAL-002 | The library shall support .NET 9 runtime. |
| REQ-QUAL-003 | The library shall support .NET 10 runtime. |
| REQ-QUAL-004 | The library shall provide type-safe APIs with nullable reference types. |
| REQ-QUAL-005 | The library shall provide comparison utilities for SPDX elements. |
